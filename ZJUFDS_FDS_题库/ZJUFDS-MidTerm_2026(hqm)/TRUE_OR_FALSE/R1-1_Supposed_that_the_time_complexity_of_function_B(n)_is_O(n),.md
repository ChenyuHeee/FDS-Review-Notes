## R1-1 Supposed that the time complexity of function B(n) is O(n), the

- **类型**: TRUE_OR_FALSE
- **分值**: 2 分
- **作者**: 何钦铭

---

Supposed that the time complexity of function B(n) is $\Theta (n)$.  Then the time complexity of the following program is $$\Theta (n^2)$$.
```
for ( i=n; i>0; i=i/2) 
    B(n);
```
