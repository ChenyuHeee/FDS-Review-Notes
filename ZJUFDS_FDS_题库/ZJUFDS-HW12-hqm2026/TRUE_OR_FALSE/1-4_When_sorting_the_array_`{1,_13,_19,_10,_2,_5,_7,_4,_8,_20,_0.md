When sorting the array ```{1, 13, 19, 10, 2, 5, 7, 4, 8, 20, 0, 15, 3, 11, 17}``` in ascending order by Shell Sort using Hibbard’s increments ($h_k=2^k-1)$, the sorting result after the first run is ```{1, 8, 19, 0, 2, 3, 7, 4, 13, 20, 10, 15, 5, 11, 17}``` and that after the second run is ```{0, 2, 3, 1, 4, 13, 5, 8, 15, 7, 10, 17, 20, 11, 19}``` .
