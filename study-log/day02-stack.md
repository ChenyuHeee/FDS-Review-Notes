## Step 1: 数组栈实现

**题目**：用数组实现栈，包含 init / push / pop / isEmpty。

### 你的实现

```c
typedef struct {
    int data[100];
    int top;
} Stack;

void init(Stack* s) {
    s->top = -1;
}

void push(Stack* s, int x) {
    s->data[++(s->top)] = x;
    // BUG记录1: 没做溢出检查，栈满时继续写会数组越界
}

int pop(Stack* s) {
    return s->data[(s->top)--];
    // BUG记录2: 没做空栈检查，空栈 pop 会读到 data[-1]
}

int isEmpty(Stack* s) {
    return s->top == -1 ? 1 : 0;
    // BUG记录3: == 本身返回 0/1，不需要 ?1:0
}
```
> ✅ 通过

### 修正后版本

```c
void push(Stack* s, int x) {
    if (s->top >= 99) return;   // 溢出保护
    s->data[++(s->top)] = x;
}

int pop(Stack* s) {
    if (isEmpty(s)) return -1;  // 空栈保护
    return s->data[(s->top)--];
}

int isEmpty(Stack* s) {
    return s->top == -1;
}
```

## Step 2: 后缀表达式求值 (Postfix Evaluation)

**题目**：求后缀 `6 2 / 3 - 4 2 * +`

**规则**：数字入栈，运算符 pop 两个、算完再 push。

### 手算过程

| 扫描 | 操作 | 栈 |
|------|------|-----|
| 6 | push 6 | [6] |
| 2 | push 2 | [6,2] |
| / | pop 2,pop 6 → 6/2=3 push | [3] |
| 3 | push 3 | [3,3] |
| - | pop 3,pop 3 → 3-3=0 push | [0] |
| 4 | push 4 | [0,4] |
| 2 | push 2 | [0,4,2] |
| * | pop 2,pop 4 → 4*2=8 push | [0,8] |
| + | pop 8,pop 0 → 0+8=8 push | [8] |

**结果：8** ✅

## Step 3: 中缀转后缀 (Infix to Postfix)

**核心规则**：数字直接输出；运算符比较优先级，栈内 >= 栈外就先弹出。

### 优先级表

| 运算符 | 栈内 ISP | 栈外 ICP |
|--------|:-------:|:-------:|
| `(` | 0 | 6 |
| `+ -` | 3 | 2 |
| `* /` | 5 | 4 |
| `)` | — | 1 |

**规则**：ICP > ISP 就入栈，否则一直 pop 直到条件满足。`)` 一直 pop 到 `(`。

### 题目 1: `1 + 2 * 3`

结果：`1 2 3 * +` ✅

### 题目 2: `(1 + 2) * 3 - 4 / (5 + 6)`

结果：`1 2 + 3 * 4 5 6 + / -` ✅

### 完整过程 (题目 2)

| 扫描 | 操作 | 栈 | 输出 |
|------|------|-----|------|
| `(` | push `(` | `[(` | |
| `1` | 输出 | `[(` | `1` |
| `+` | ICP=2 > ISP=0, push | `[(, +]` | `1` |
| `2` | 输出 | `[(, +]` | `1 2` |
| `)` | pop 直到 `(` | `[]` | `1 2 +` |
| `*` | push | `[*]` | `1 2 +` |
| `3` | 输出 | `[*]` | `1 2 + 3` |
| `-` | ICP=2 < ISP=5, pop `*`; push `-` | `[-]` | `1 2 + 3 *` |
| `4` | 输出 | `[-]` | `1 2 + 3 * 4` |
| `/` | ICP=4 > ISP=3, push `/` | `[-, /]` | `1 2 + 3 * 4` |
| `(` | push `(` | `[-, /, (]` | `1 2 + 3 * 4` |
| `5` | 输出 | `[-, /, (]` | `1 2 + 3 * 4 5` |
| `+` | push `+` | `[-, /, (, +]` | `1 2 + 3 * 4 5` |
| `6` | 输出 | `[-, /, (, +]` | `1 2 + 3 * 4 5 6` |
| `)` | pop `+`, pop `(` | `[-, /]` | `1 2 + 3 * 4 5 6 +` |
| 结束 | pop 全部 | `[]` | `1 2 + 3 * 4 5 6 + / -` |
