## Step 1: 插入排序 (Insertion Sort) — 复习

```c
void insertionSort(int A[], int N) {
    int i, j, tmp;
    for (i = 1; i < N; i++) {
        tmp = A[i];
        for (j = i; j > 0 && A[j-1] > tmp; j--)
            A[j] = A[j-1];
        A[j] = tmp;
    }
}
```

## Step 2: 希尔排序 (Shellsort)

### 核心理解

插入排序只是 gap=1 的特殊情况。希尔排序在外层包一个 gap 循环，把"相邻右移"变成"间隔 gap 右移"。

### BUG 记录

**BUG1: while(gap!=1) 吞了最后一轮**
```c
while (gap != 1) { gap /= 2; }  // gap=2→1 时 while 判断 gap!=1，跳出！gap=1 这轮没跑
// 正确: while (gap > 0)
```

**BUG2: i += gap 丢掉了大部分元素**
```c
for (i = 1; i + gap <= N; i += gap)  // 只处理了一组！其他组的元素全跳过
// 正确: for (i = gap; i < N; i++)   // 每次+1，所有元素都处理到
```

**BUG3: 内层 for 里声明的 j 在外面用不了**
```c
for (int j = i; ...; ...)  // j 作用域只在这个 for 里
A[j] = tmp;                 // 编译错误：j 未定义
// 正确: j 提出来声明
```

**BUG4: 函数名拼错 shellshort → shellSort**

**BUG5: i <= N 越界** → 应该是 `i < N`

### 最终代码

```c
void shellSort(int A[], int N) {
    int gap = N, j, tmp;
    while (gap > 0) {
        gap /= 2;
        for (int i = gap; i < N; i++) {
            tmp = A[i];
            for (j = i; j >= gap && A[j-gap] > tmp; j -= gap)
                A[j] = A[j-gap];
            A[j] = tmp;
        }
    }
}
```

## Step 3: 堆排序 (Heapsort)

### Q&A: 原地堆排序 vs 堆 ADT

| | 堆 ADT | 原地堆排序 |
|------|--------|----------|
| 存储 | `Heap* h` 结构体 | 直接用 `int A[]` |
| 索引 | 从 1 开始 | 从 0 开始 |
| 孩子 | `2*i`, `2*i+1` | `2*i+1`, `2*i+2` |
| 父 | `i/2` | `(i-1)/2` |
| 类型 | 最小堆 | **最大堆**（根最大，甩到末尾） |

### BUG 记录

**BUG1: BuildHeap 写成传 Heap 结构体**
```c
Heap* h;  // 没分配内存，野指针
BuildHeap(h, A, N);  // 原地堆排序不需要额外的堆结构体！
```

**BUG2: BuildHeap 拷数组时偏移错**
```c
h->data[i] = *(arr + i);  // arr[0] → data[0]，但堆从 data[1] 开始
// 正确: h->data[i] = arr[i-1];
```

**BUG3: 循环条件 j > 1 漏了根**
```c
for (j = N/2; j > 1; j--)  // 漏了 j=1
// 正确: j >= 1
```

**BUG4: 循环条件 i < N 漏了最后一个元素**
```c
for (i = 1; i < N; i++)  // 只拷了 N-1 个元素
// 正确: i <= N
```

**BUG5: percDown 里写成堆结构体的语法**
```c
if (size > child+1 && h->data[child] > h->child[child+1])  // size/h->data/h->child 都不存在
// 正确: if (N > child+1 && A[child] < A[child+1])  ← 最大堆找较大孩子
```

**BUG6: 堆排序阶段只会建堆，不会排序**
```c
// 忘了 for (i=N-1; i>0; i--) { swap + percDown }
```

### 最终代码

```c
void percDown(int A[], int i, int N) {
    int child, tmp = A[i];
    while (2*i+1 < N) {
        child = 2*i + 1;
        if (child+1 < N && A[child] < A[child+1])   // 最大堆：找较大孩子
            child++;
        if (tmp >= A[child]) break;
        A[i] = A[child];
        i = child;
    }
    A[i] = tmp;
}

void heapSort(int A[], int N) {
    int i;
    for (i = N/2 - 1; i >= 0; i--)      // BuildMaxHeap
        percDown(A, i, N);
    for (i = N - 1; i > 0; i--) {        // 排序
        int tmp = A[0]; A[0] = A[i]; A[i] = tmp;
        percDown(A, 0, i);
    }
}
```

## Step 4: 归并排序 (Mergesort)

### Q&A: tmp 参数要不要传？

不需要传，在函数里临时 malloc/free。

### Q&A: k 从 0 还是从 left 开始？

tmp 是局部数组，从 0 开始。拷回时 `A[left+p] = tmp[p]`。

### Q&A: while 循环终止条件

```c
while (i <= mid && j <= right)   // 只要两段都没走完
```

### Q&A: 参数 left/mid/right 的含义

```
[1, 4, 6 | 2, 3, 5]
 ↑    ↑   ↑       ↑
left mid mid+1   right
```

### BUG 记录

**BUG1: n 声明在 malloc 后面 → 编译错误**

### 最终代码

```c
void merge(int A[], int left, int mid, int right) {
    int i = left, j = mid + 1, k = 0;
    int n = right - left + 1;
    int* tmp = (int*)malloc(n * sizeof(int));
    while (i <= mid && j <= right) {
        if (A[i] < A[j]) tmp[k++] = A[i++];
        else             tmp[k++] = A[j++];
    }
    while (i <= mid)   tmp[k++] = A[i++];
    while (j <= right) tmp[k++] = A[j++];
    for (int p = 0; p < n; p++)
        A[left + p] = tmp[p];
    free(tmp);
}

void mergeSort(int A[], int left, int right) {
    if (left >= right) return;
    int mid = (left + right) / 2;
    mergeSort(A, left, mid);
    mergeSort(A, mid + 1, right);
    merge(A, left, mid, right);
}
```
